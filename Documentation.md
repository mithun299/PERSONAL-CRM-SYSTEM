# 1. INTRODUCTION

The **Personal CRM System** is a web application developed as part of the **Full Stack Web Development Internship** at **CODTECH IT SOLUTIONS PRIVATE LIMITED**. It enables users to efficiently manage personal and professional contacts through an intuitive and responsive interface.

The application allows users to add, edit, search, and delete contact information while using **Browser Local Storage** to ensure data persistence without requiring a backend database.

---

# 2. OBJECTIVE

The objectives of this project are:

* To maintain customer and personal contact records.
* To perform CRUD (Create, Read, Update, Delete) operations.
* To provide quick access to stored contacts.
* To implement Browser Local Storage for persistent data storage.
* To create a responsive and user-friendly web application.
