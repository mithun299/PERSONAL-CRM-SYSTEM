# PROJECT DESCRIPTION

The **Personal CRM System** is a web-based application developed as part of the **Full Stack Web Development Internship** at **CODTECH IT SOLUTIONS PRIVATE LIMITED**. The application enables users to efficiently manage personal and professional contacts through a clean, responsive, and user-friendly interface.

Users can add, edit, delete, search, and manage contact information, with all data securely stored using the browser's Local Storage.

---

# PROJECT SCOPE

The project is designed to:

* Manage personal and professional contacts.
* Store contact information digitally.
* Search contacts quickly and efficiently.
* Perform CRUD (Create, Read, Update, Delete) operations.
* Demonstrate JavaScript Local Storage functionality.
* Develop a responsive web application using modern front-end technologies.
